Vulnerability: Path Traversal / Directory Traversal

  • The server constructs a file path from your input and reads it. It doesn't sanitize ../ sequences.
  • Try entering: ../secret.txt or ../../etc/passwd
  • The key payload that triggers the flag is any path containing ../ or ..\
  • The server detects directory traversal sequences and returns the flag as the "file content".
