Vulnerability: OS Command Injection

  • The server passes your IP directly to an OS shell command like: ping [YOUR_INPUT]
  • Append additional commands using shell operators: ;, &&, or |
  • Try: 127.0.0.1; ls or 127.0.0.1 && echo hello
  • The server also checks for: cat, ls, dir, echo commands in the payload.
  • A valid payload example: 127.0.0.1; cat /etc/passwd
