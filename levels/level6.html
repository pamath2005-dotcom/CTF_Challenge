Vulnerability: Insecure Direct Object Reference (IDOR)

  • You start viewing profile ID 1 (a regular user). You are "user 1".
  • There is an admin user with a different ID. The server does not verify you are authorized to view other profiles.
  • Simply change the ID number in the input from 1 to 2 and click Fetch Profile.
  • The admin profile's private_flag field contains the flag.
