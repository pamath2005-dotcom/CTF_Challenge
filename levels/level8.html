Vulnerability: Insecure Deserialization / Token Tampering

  • The default token is eyJyb2xlIjoidXNlciJ9 — decode it with Base64 to see: {"role":"user"}
  • You can decode Base64 in your browser console: atob("eyJyb2xlIjoidXNlciJ9")
  • Create a new token with {"role":"admin"} encoded in Base64.
  • In your browser console: btoa('{"role":"admin"}') gives you the forged token.
  • Paste the new token into the input field and click Authenticate.

Identity Verification

MEDIUM +100 EXP

The server issues a Base64-encoded token to verify who you are. Can you alter your identity to become an 'admin'? Currently your token says you are a 'user'. The server trusts whatever token you submit — no signature verification!
