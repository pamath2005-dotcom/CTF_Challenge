Vulnerability: Cookie Manipulation / Broken Access Control

  • Step 1: Click "Acquire Guest Identity" to set a role=user cookie.
  • Step 2: Open DevTools → Application tab → Cookies → find the role cookie.
  • Step 3: Change the cookie value from user to admin.
  • Step 4: Click "Access Admin Panel" — the server reads the cookie raw and grants access!
