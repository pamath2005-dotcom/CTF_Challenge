Vulnerability: XML External Entities (XXE)

  • The server parses XML you send without disabling external entity processing.
  • The payload is already pre-loaded — it defines an external entity pointing to file:///etc/passwd.
  • The key structure is: <!DOCTYPE> → <!ENTITY xxe SYSTEM "file://..."> → reference &xxe; in a tag.
  • Simply click Submit XML with the default payload — the server detects it and returns the mock file contents including the flag.
  • The flag will be visible in the response output below.

XML Order Processor

HARD +100 EXP

The server accepts XML for processing orders. The XML parser does not disable external entity processing. Try crafting a payload that references an external SYSTEM entity pointing to a local file like file:///etc/passwd.
