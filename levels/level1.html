Vulnerability: Information Exposure / Security Misconfiguration

  • Click the "Load System Config" button and watch what happens in the browser network tab (F12 → Network).
  • The page only displays some of the returned data, but the full API response is visible in the Network tab.
  • Look at the raw JSON response from /api/level1/config.
  • The flag will be inside the admin_token field.

Mission Brief

EASY +100 EXP

The system administrators left the server configuration exposed on the network. They claim it's "security through obscurity". Check how this page loads its data — not everything shown to the UI is all the data being returned.
